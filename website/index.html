📚 Theory & Background

Newton-Raphson Method

The Newton-Raphson method is an iterative algorithm for finding successively better approximations to the roots (or zeros) of a real-valued function. Given a function f(x) and its derivative f'(x), the method uses the formula:

xn+1 = xn - f(xn)/f'(xn)

Key Points:

  • Requires the derivative of the function
  • Quadratic convergence (very fast when close to root)
  • May fail if derivative is zero or nearly zero
  • Sensitive to initial guess selection

Fixed Point Iteration

Fixed Point Iteration finds a solution to x = g(x) by repeatedly applying the function. To solve f(x) = 0, we rearrange it into the form x = g(x) and iterate:

xn+1 = g(xn)

Key Points:

  • Simpler than Newton-Raphson (no derivative needed)
  • Converges if |g'(x)| < 1 near the fixed point
  • Typically linear convergence (slower than Newton-Raphson)
  • Convergence depends heavily on the choice of g(x)
